# RNwNJS (React-Native with NextJS)

This monorepo is a starter for an Expo + Next.js app using [NativeWind](https://nativewind.dev) for its styling & [Solito](https://solito.dev) for navigation.

- **React Native** (v18)
- **Expo** (SDK 47)
- **NextJs** (v13)
- **Solito** (v2)
- **NativeWind** (v2)
